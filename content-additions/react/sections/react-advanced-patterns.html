Advanced Patterns

Design patterns for building scalable, reusable, and type-safe components.

Component Composition & Slot Pattern

Instead of passing dozens of props to a component, use composition to keep components clean and flexible.

  • Composition: Pass components as `children` or explicit props.
  • Slots: Define specific "plugs" where child components can be inserted, allowing the parent to control the layout while the consumer controls the content.
// Slot Pattern
function Modal({ header, body, footer }) {
  return (
    <div class="modal">
      <div class="h">{header}</div>
      <div class="b">{body}</div>
      <div class="f">{footer}</div>
    </div>
  );
}

// Usage
<Modal 
  header={<h1>Welcome</h1>} 
  body={<LoginForm />} 
  footer={<Button>Close</Button>} 
/>
