React 19 Deep Dive

The biggest shift in React history: Compiler, Server Components, and native Actions.

React Compiler (Experimental)

React 19 introduces a build-time compiler that automatically memoizes your code. It ensures that components re-render only when their data truly changes.

  • Goodbye `useMemo` & `useCallback`: The compiler analyzes your dependency graph and adds auto-memoization, making these hooks mostly redundant.
  • Zero-Config: Once enabled, it optimizes your entire application without manual intervention.
  • Strict Rules: Requires strictly following React's rules (no mutating props or state directly) to work correctly.
Server Components (RSC) & Actions

RSCs allow components to render on the server, potentially removing all their JavaScript from the client bundle.

  • Server Actions: Use `use server` to define functions that run on the server, callable directly from your client-side forms.
  • useFormStatus & useActionState: New hooks to handle the pending state and response of Server Actions without manual `isLoading` trackers.
  • Streaming: Components can stream their content to the client as soon as data (like DB queries) is ready.
// Server Action (actions.js)
'use server'
export async function updateUsername(formData) {
  const name = formData.get('name');
  await db.user.update({ name });
}

// Client Component
function ProfileForm() {
  const { pending } = useFormStatus();
  return (
    <form action={updateUsername}>
      <input name="name" />
      <button disabled={pending}>Update</button>
    </form>
  );
}
The `use()` Hook & useOptimistic

React 19 makes asynchronous logic feel synchronous within your components.

  • use(): Can be used to read a Promise or Context. Unlike standard hooks, it can be used inside `if` statements or loops.
  • useOptimistic: Allows you to immediately update the UI with an expected result while a server action is still in flight.
function MessageList({ messages, sendMessage }) {
  const [optimisticMessages, addOptimisticMessage] = useOptimistic(
    messages,
    (state, newMessage) => [...state, { text: newMessage, sending: true }]
  );

  return (
    <div>
      {optimisticMessages.map(m => <div>{m.text}</div>)}
      <button onClick={() => {
        addOptimisticMessage("Hello!");
        sendMessage("Hello!");
      }}>Send</button>
    </div>
  );
}
