Anchor Positioning API

Position elements relative to any other "anchor" without shared parents or JS libraries:

.anchor { anchor-name: --my-btn; }

.popover {
  position: absolute;
  position-anchor: --my-btn;
  bottom: anchor(top);
  left: anchor(center);
}
  • Native Popovers: Combined with `popover` attribute for top-layer rendering.
  • Fallback Logic: Use `position-try-options` for safe repositioning.
EXPERT

View Transitions API

Seamlessly animate between DOM states (or pages) without complex JS libraries:

function updateUI() {
  document.startViewTransition(() => {
    // Perform DOM modifications
    renderNewPage(); 
  });
}
  • Pseudo-elements: Automatically creates snapshots `::view-transition-group`.
  • Cross-Document: Supported in 2024/2025 for standard multi-page sites.
ADVANCED

Cascade Layers (@layer)

Architectural control over specificity, solving the 3rd-party override library wars:

@layer base, utility, components;

@layer utility {
  .p-10 { padding: 10px !important; }
}

@layer base {
  /* Won't be overridden by higher layers 
     despite specificity */
}
EXPERT

Relative Color Syntax (RCS)

Derive dynamic colors natively from existing variables:

:root { --primary: #3498db; }

.btn {
  /* Extract R, G, B and add 50% alpha */
  background: rgb(from var(--primary) r g b / 0.5);
  /* Or oklch adjustments */
  border-color: oklch(from var(--primary) calc(l + 0.1) c h);
}
