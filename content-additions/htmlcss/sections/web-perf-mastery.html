Interaction to Next Paint (INP)

The modern metric representing total interaction latency. INP tracks every click, tap, and keyboard interaction throughout the entire session:

  • Input Delay: Time between user interaction and event handler start (minimize main thread blocking).
  • Processing Time: Time spent inside the event handler (optimize logic, use `setTimeout(0)` or `isInputPending`).
  • Presentation Delay: Time spent painting the frame (avoid layout thrashing / heavy CSS).

Image & Asset Optimization

Modern formats and strategies for minimizing cumulative layout shifts:

  • AVIF & WebP: Native compression over 50% better than JPEG.
  • Aspect Ratio: Reserve space before images load to prevent layout shifts.
  • Source Set: Use `srcset` and `sizes` to deliver the perfect pixel density for the device.
