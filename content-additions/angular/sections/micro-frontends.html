Micro-Frontends

Scaling applications across multiple independent teams and codebases.

Module Federation (Webpack) / Native Federation (Vite/Esbuild)

Micro-frontends allow you to split a monolithic Angular app into smaller, independently deployable applications. These applications load each other's compiled code at runtime.

  • Host Application (Shell): The main app that boots up, handles global routing, and loads remote modules.
  • Remote Application: A standalone application that exposes parts of itself (components or routes) to be consumed.
  • Shared Dependencies: Federation allows you to declare dependencies (like `@angular/core` or RxJS) as singletons, so the browser doesn't download the Angular framework 5 times.

Because Angular 17+ moved away from Webpack towards Esbuild/Vite, the standard tool has become Native Federation, which uses browser-native Import Maps rather than Webpack-specific logic.

// Host routing loading a remote Micro-Frontend via Native Federation
export const routes: Routes = [
  {
    path: 'flights',
    loadChildren: () =>
      loadRemoteModule('flights', './Component')
        .then(m => m.FlightsComponent)
  }
];
Angular Elements (Web Components)

Angular Elements allows you to package an Angular component as a standard Custom Element. This is the ultimate "agnostic" micro-frontend strategy.

  • Interoperability: Can be used in React, Vue, or vanilla JS without the host knowing it's Angular.
  • Isolated Styles: Uses Shadow DOM for style encapsulation.
  • Challenge: The initial bundle size is larger because it must include the Angular runtime inside each element (unless using shared vendors).
Shared State in Micro-Frontends

Sharing state between independent apps is the biggest challenge in MFE architecture.

  • Browser Events: Using `Window.dispatchEvent` for loose coupling.
  • Shared Libraries: A shared NgRx/SignalStore library that both apps import (must be configured as a singleton dependency).
  • Local Storage / Cookies: For persistent shared data like auth tokens.
