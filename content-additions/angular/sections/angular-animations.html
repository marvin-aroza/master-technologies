Animations

Building rich, performant UI motion using Angular's DSL on top of the Web Animations API.

Animation DSL API

Angular provides a domain-specific language (DSL) for defining animations inside component metadata. It uses `BrowserAnimationsModule` behind the scenes.

  • trigger(): Defines an animation name that binds to an element via `[@myTrigger]`.
  • state(): Defines specific styling points (e.g., "open", "closed").
  • transition(): Defines how to animate between states (e.g., `'open => closed'`). Special states include `void` (not attached to DOM) and `*` (any state). Fast syntax: `':enter'` (`void => *`) and `':leave'` (`* => void`).
  • animate(): Specifies duration, delay, and easing (e.g., `animate('200ms ease-out')`).
  • style(): Specifies the CSS properties to animate.
import { trigger, state, style, animate, transition } from '@angular/animations';

@Component({
  template: `
Toggle Me
`,
  animations: [
    trigger('openClose', [
      state('open', style({ height: '200px', opacity: 1, backgroundColor: 'yellow' })),
      state('closed', style({ height: '100px', opacity: 0.5, backgroundColor: 'green' })),
      transition('open => closed', [ animate('1s') ]),
      transition('closed => open', [ animate('0.5s') ]),
    ]),
  ]
})
export class AnimatedComponent { isOpen = true; }
Advanced Animation Control

For more complex orchestrations, Angular animations provide grouping and querying:

  • query(): Finds inner elements to animate them inside the parent's transition.
  • stagger(): Delays the start time of multiple queried items (great for animating lists).
  • group(): Runs multiple animation steps in parallel.
  • sequence(): Runs steps sequentially (default behavior of arrays).
  • animateChild(): Tells Angular to execute child component animations, which would otherwise be blocked by the parent animation.
trigger('listAnimation', [
  transition('* => *', [
    query(':enter', [
      style({ opacity: 0, transform: 'translateY(-15px)' }),
      stagger('50ms', [
        animate('300ms ease-out', style({ opacity: 1, transform: 'none' }))
      ])
    ], { optional: true })
  ])
])
